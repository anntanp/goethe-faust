# Inputs

## data/items-all-goethe-faust.json

- **Path**: `goethe-faust/data/items-all-goethe-faust.json`
- **Format**: JSONL (one JSON object per line, not a JSON array)
- **Provenance**: DDB EDM export (Goethe-Faust corpus); built by `fetch-items.sh`
- **Size**: 115,432 records
- **Key fields**:
  - `edm.RDF.ProvidedCHO.dcType` ‚Äî dc:type node: `{resource, lang, $}` or list thereof; text value at `$`
  - `edm.RDF.Concept[]` ‚Äî list of vocabulary nodes; each has `about` (URI) and `prefLabel` (`{lang, $}` or null)
    - Mediatype nodes: `about` starts with `http://ddb.vocnet.org/medientyp/`
    - Sector nodes: `about` starts with `http://ddb.vocnet.org/sparte/`
    - GND concept nodes: `about` starts with `https://d-nb.info/gnd/`; `prefLabel.$` matches dc:type string
  - `edm.RDF.ProvidedCHO.hierarchyType` ‚Äî htype code (e.g. `htype_001`)
  - `edm.RDF.WebResource[].about` ‚Äî WebResource URI (target of `edm:isShownBy` / `edm:hasView`)
  - `provider-info.domains[0]` ‚Äî sector IRI (redundant with Concept)
- **Encoding**: UTF-8

## data/items/*.json

- **Path**: `goethe-faust/data/items/`
- **Format**: JSON, one file per DDB item (115,437 files; 4 empty/corrupt)
- **Provenance**: DDB EDM export; same records as JSONL above, one file per record
- **Encoding**: UTF-8

## output/dctype_frequency_all.csv

- **Path**: `goethe-faust/output/dctype_frequency_all.csv`
- **Format**: CSV, columns: `mediatype, sector, dc_type_de, count`
- **Provenance**: Generated by `scripts/count_dctype_by_mediatype.py` from `data/items/*.json`
- **Size**: 1,386 rows
- **Volume by mediatype**:
  - mt001 (Audio): 476 records, 8 unique dc:types
  - mt002 (Photo): 20,229 records, 695 unique dc:types
  - mt003 (Text): 52,247 records, 169 unique dc:types
  - mt005 (Video): 96 records, 10 unique dc:types
  - mt007 (Not Digitized): 42,360 records, 310 unique dc:types
